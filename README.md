## 📌 Project Overview
Interactive safety monitoring dashboard for 
clinical trial data analysis. Built using Python 
and Plotly for real-time adverse event tracking.

## 📊 Dashboard Features
- AE Severity Distribution (Pie Chart)
- Average AE Count by Site (Bar Chart)
- Dropout Rate by Treatment Group
- Age Distribution by AE Severity

## 🎯 Clinical Significance
- Real-time safety signal detection
- Site performance monitoring
- Patient dropout analysis
- Treatment group comparison

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- Plotly (Interactive Visualizations)
- Google Colab

## 📈 Dataset
- 1000 patients across 4 clinical sites
- 10 clinical variables
- AE, SAE, dropout tracking

## 🚀 How to Run
1. Clone repository
2. pip install plotly pandas
3. Open notebook in Colab
4. Run all cells
